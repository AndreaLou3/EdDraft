# EdDraft

Tool for templating and posting EdStem threads

## Requirements

- [edapi](https://github.com/smartspot2/edapi) >=0.1.0
- [Jinja2](https://jinja.palletsprojects.com/en/stable/) >=3.1.6

### Setup

1. Install dependencies
    ```js
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -e .
    ```
